# LearnSphere 🎓

A comprehensive Django-based student dashboard simulating a real-life academic system.

## 🔍 Features

- Secure login & logout
- Dynamic student/instructor profiles
- Class schedules, grades, assignments, announcements
- Responsive Bootstrap 5 UI
